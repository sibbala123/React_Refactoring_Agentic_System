LAYER 2 FastAPI Server Python, server/app.py, Port 7432
API Endpoints
Method Endpoint Description Called From
GET /health Returns server status and version. Polled every 250ms on startup. serverManager.ts
POST /scan Runs detect_smells.py, enriches line ranges, applies severity rules, returns smell list as JSON. runScan()
POST /fix Accepts smell tasks, creates a job, launches the LangGraph pipeline. Returns a job ID. fixSelected cmd
GET /progress/{job_id} Server-Sent Events stream. Emits per-smell status changes as the pipeline progresses. SSE reader
POST /cancel/{job_id} Sets a cancellation flag. In-flight pipeline runners check this flag and abort early. cancelFix cmd
POST /revert Restores the file from a backup created before the fix was applied. revertSmell cmd
GET /original Runs git show HEAD on the file for the diff view. Returns 404 if workspace is not a git repo. viewDiff cmd

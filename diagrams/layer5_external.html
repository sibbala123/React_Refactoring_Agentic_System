LAYER 5 External Dependencies Runtime requirements
OpenAI API
GPT-4o powers the Classify, Plan, Edit, and Critique pipeline nodes. The API key is read from VS Code settings or the OPENAI_API_KEY environment variable.
GPT-4o Chat Completions
Git CLI
Used by the /original endpoint to retrieve pre-fix file content via git show HEAD. Returns a 404 response gracefully when the workspace is not a git repository.
git show subprocess
Node.js
Required to run the bundled ReactSniffer CLI. Invoked via npx subprocess from detect_smells.py. The ReactSniffer package is included in the VSIX bundle.
Node.js 16+ npx
Python 3.8+
Host machine runtime for the FastAPI server and the entire LangGraph pipeline. Detected automatically via the ms-python VS Code extension, then by searching PATH.
FastAPI LangGraph uvicorn
File System
The server reads and writes source files directly. A backup of the original file content is saved before any fix is applied, enabling the /revert endpoint to restore prior state.
read and write backups

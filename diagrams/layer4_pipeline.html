LAYER 4 LangGraph Fix Pipeline agentic_refactor_system/langgraph_pipeline/
Pipeline Node Graph
Classify
classify.py
Decides if the smell is fixable or should be skipped
Plan
plan.py
Generates a step-by-step fix strategy using GPT-4o
Edit
edit.py
Applies code changes to the file using the plan
Verify
verify.py
Reruns ReactSniffer to confirm the smell is resolved
Critique
critique.py
LLM code review, assigns a quality score
Finalize
finalize.py
Writes the file and emits an SSE status event
↩ If critique score is too low, routes back to Plan for another attempt
Supporting Modules
state.py
LangGraph state schema. Carries smell metadata, plan, edited code, verify result, critique score, and retry count between nodes.
runner.py
Async executor that runs the graph per smell, emits progress events over SSE, and checks the cancellation flag.
tactics.py
Maps each smell type to a recommended fix strategy, which is fed into the Plan node as additional context.
